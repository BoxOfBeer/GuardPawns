# Действие: fix_cs0117_cs0103_pawnagent

## Цель
Устранить ошибки компиляции CS0117 и CS0103 в `SpaceBall/Core/PawnAgent.cs` после изменений подсистемы сэмплинга поверхности.

## Контекст
- Входные данные: лог сборки с ошибками:
  - `WorldConstants` не содержит `PawnVisualSurfaceOffset`.
  - `WorldConstants` не содержит `PawnModelBaseOffset`.
  - `_heightmap` не существует в текущем контексте.
- Ограничения: правка только по месту ошибки, без изменения UI и без изменения внешнего поведения симуляции.
- Ожидаемый артефакт: компилируемый `PawnAgent` с корректными ссылками на актуальные поля/источники данных.

## План
1. Проверить текущие определения в `PawnAgent` и `WorldConstants`.
2. Заменить несуществующие константы `WorldConstants` на локальные константы `PawnAgent`.
3. Убрать обращение к удалённому `_heightmap` и использовать флаг наличия данных в `SurfaceSampler`.
4. Проверить сборку проекта.

## Изменения (по файлам)
- `SpaceBall/Core/PawnAgent.cs`

- Локальные константы уже объявлены в `PawnAgent`; их использование сохраняет текущую задумку смещений без расширения `WorldConstants`.
